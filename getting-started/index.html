Getting Started

This is a short tutorial on how to get started with Feram.

Step 1: Activate Feram

Feram is a GitHub app and can be easily enabled for your repositories.

First, go to github.com/apps/feram and activate Feram for at least one repository.

Whenever you create a new pull request in one of the activated repositories, Feram automatically checks the code of the pull request and posts a warning if any bugs were found:

Step 2: Select Patches

When you open the "Details" link, you will get redirected to the corresponding "code check" on feram.io:

There you simply select which patches you want to apply to your code and click submit.

Step 3: Merge Pull Request

Feram then commits all the changes directly into the pull request branch and you are ready to finally merge the pull request!

Celebrate

Congratulations, you have effortlessly improved your code!

From now on every pull request in you activated repositories gets automatically checked by Feram.
