Getting Started on Bitbucket

This is a short tutorial on how to get started with Feram on Bitbucket.

Step 1: Activate Feram

Feram is a Bitbucket integration and can easily be enabled for your repositories.

  1. Go to bitbucket.org/account/user/YOUR_USERNAME/addon-management
  2. Click on "Install add-on from URL"
  3. Enter following URL: https://app.feram.io/bitbucket/ (Including trailing slash)

Whenever you create a new commit in one of the activated repositories now, Feram automatically starts to check the code of the repo:

And updates the status once the code check finishes:

You can click on the build icon to get redirected to the corresponding "code check" on feram.io.

This works even if the code check is still running as we're updating the corresponding page in real time.

Step 2: Select Patches

On the code check page you can select which patches you want to apply to your code.

Step 3: Submit

At the end you can submit the patches and Feram commits all of them directly into the current branch. If it was part of a merge request the branch can then be merged!

Celebrate

Congratulations, you have effortlessly improved your code!

From now on every commit in you activated repositories gets automatically checked by Feram.
