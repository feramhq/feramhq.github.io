General

What is Feram? #

Feram is a Service which finds bugs in your merge requests and automatically generates patches for those bugs. You then can step through the patches and decide which ones you want to apply and they are automatically committed to your merge request.

How does Feram work? #

Feram is a GitHub App which seamlessly integrates in your GitHub merge request workflow.

Every time a new merge request is opened, or new commits are pushed into an existing merge request Feram gets notified and automatically downloads the latest changes. It then uses state of the art open source tools to detect bugs in your code. (Check out the full list on our features page.) If any bugs were found we block the merge request and post a link to feram where you can select which patches you want to apply and which you want to discard.

After making your selection we commit the relevant patches and push them onto your merge request branch. The merge request can then be merged.

For a detailed tutorial on how to get started check out the getting started page.
